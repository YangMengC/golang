常量：
    `const identifier [type] = value`
    多个相同类型的声明可以简写为：
    `const c_name1, c_name2 = value1, value2`

左移运算符 << 是双目运算符。
    左移 n 位就是乘以 2 的 n 次方。 
    其功能把 << 左边的运算数的各二进位全部左移若干位，由 << 右边的数指定移动的位数，高位丢弃，低位补 0。

右移运算符 >> 是双目运算符。
    右移 n 位就是除以 2 的 n 次方。 
    其功能是把 >> 左边的运算数的各二进位全部右移若干位， >> 右边的数指定移动的位数。
