`<<`表示在二进制数字中向左移位如：
    `3<<1`意思就是在11的基础上向左移动一位变成110，所以该式=6

iota属于一个比较特殊的常量，当在const初始化开始，iota即开始从0递增
    ``` 
        const (
            n = 12 //此时iota已经开始计数
            i = iota //1
            j = iota //2
            k
            l
            )
    ```
    此处的k与l即与j一样，即当这样书写时，等同于
    ```
    const (
            n = 12 //iota=0
            i = iota //1
            j = iota //2
            k = iota //3
            l = iota //4
            )
    ```
